C7	Skill hit changes caused skill delay on hits.
B8	Detailed description of issue
C8	All classes have seen a discernible decrease in skill hits with the 19th of June 2023 patch notes, but Valkyrie appears to have experienced a major bug as a result. In particular, there are now lag or delays on a few skills when using them against opponents or mobs. This delay is especially noticeable when using the Divina Inpulsa, Blitz Stab, and Promptness skills. The combat flow is significantly hampered by these delays, which makes it challenging to execute combos and react skillfully in combat.
